## Ejemplos de Automator

### Redimensionar imágenes con Sharp CLI

Copia el archivo los archivos *workflow* a la carpeta `~/Library/Services/`

Ahora puedes usar el servicio desde Finder, seleccionando las imágenes y haciendo clic derecho y eligiendo "Acciones rápidas" > "Redimensionar imágenes con Sharp CLI"

Para que funcione correctamente, debes abrirlos (doble click, se abrirá automator) y darle a GUARDAR.
